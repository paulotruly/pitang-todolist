# To do list + Pomodoro
Uma aplicação de produtividade de alto desempenho que combina a técnica 'Pomodoro' com uma lista de tarefas dinâmica.

## Demonstração do projeto

https://github.com/user-attachments/assets/87bc66d9-84d9-402e-b91b-201337c59625

## Tecnologias utilizadas

* **React**: Biblioteca para construção de interfaces baseadas em componentes e estado reativo.
* **Tailwind CSS**: Permite uma UI responsiva e moderna com classes utilitárias, facilitando temas dinâmicos.
* **TypeScript**: Garante segurança de tipos, evitando erros comuns ao manipular IDs de setInterval e estados de modos. 

## Como executar o projeto  

1. Clone o repositório:  
```bash
git clone https://github.com/paulotruly/pitang-todolist/
```
2. Certifique-se de ter o Node.js instalado e baixe as dependências:  
```bash
npm install
```
3. Execute o projeto:  
```bash
npm run dev
```
